# Screenshot Instructions

1. Run the app: npm run dev
2. Open http://localhost:5173 in your browser
3. Take screenshots of different screens:
   - Dashboard (empty and with participants)
   - Each theme in action
   - Game complete screen
4. Save them as PNG files in the screenshots/ directory
5. Commit and push the images

Example screenshot names:
- dashboard.png
- base-theme-active.png  
- squid-game-theme.png
- game-complete.png
